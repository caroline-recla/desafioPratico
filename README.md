
# ModalGR - Processo de Formação 2024

Etapa Técnica - Composto por três desafios práticos, sendo eles: 
    1. Senha Criptografada
    2. Aniversariantes do Mês 
    3. Empréstimo de Funcionários


## Stack utilizada

**Back-end:** Java, JDK 17.0.1
**Motivo da Escolha:** Por ser uma das linguagens mais utilizadas e ser uma linguagem fortemente tipada, foi feito a opção de utilizá-la na resolução dos exercícios propostos. Um dos motivos que corroborou com isso sendo uma das primeiras linguagens aprendida e uma de minhas preferidas.
